1.1 静态分析在什么位置进行

描述： 源码经过词法分析，语法分析，语义分析后再翻译为IR，基于IR进行Static Analysis

IR ：intermediate representation ，静态分析中常使用三地址码。

1.2 AST和IR 对比

AST

  • 接近语法结构
  • language dependent
  • 缺少控制流信息
  • 适合快速类型检验

IR

  • 接近机器码
  • language independent
  • 包含控制流信息
  • 通常用于静态分析

1.2.1 三地址码

3-Address Code（3AC）

指令右边最多一个操作符

3AC的一些形式

  • x = y bop z bop:二进制数学数学或逻辑操作符
  • x = uop y uop: 一元操作符
  • x = y
  • goto L
  • if x goto L
  • if x rop y goto L rop: 关系运算符 ＞＜＝

1.4.2 Basic Blocks (BB)

Basic Blocks 是具有以下属性且最大的连续三地址指令序列

  1. 只有一个入口：块中的第一条指令
  2. 只有一个出口：块的最后一条指令
how to build BB

  1. 确定所有的入口

    • 程序的第一条指令是入口
    • 所有跳转的target是入口
    • 所有跳转指令的下一条指令是入口

  2. 确定Basic Blocks

    • 已经确定所有入口，那么一个入口到下一个入口就是一个BB
